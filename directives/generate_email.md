# Generate Email Directive

## Objective
Create personalized cold email drafts based on enriched lead data.

## Input
- Enriched lead profile
- Email templates
- Personalization rules
- Brand voice guidelines

## Process
1. Select appropriate email template
2. Personalize subject line with lead data
3. Customize opening based on company context
4. Incorporate relevant pain points/solutions
5. Add specific technology/industry references
6. Include clear call-to-action
7. Validate email length and readability
8. Check for spam trigger words

## Output
- Personalized email draft
- Subject line variants (A/B testing)
- Personalization tokens used
- Spam score
- Readability score

## Error Handling
- Missing personalization data → Use fallback template
- High spam score → Revise and regenerate
- Template errors → Log and use default
- Length violations → Trim while preserving key points

## Success Criteria
- Email is personalized with at least 3 data points
- Spam score < 5.0
- Readability grade level 8-10
- Clear value proposition and CTA
- No placeholder text remaining
